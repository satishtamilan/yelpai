üì± Quick Start (3 Steps)

Step 1: Get Your Yelp API Key

  1. Visit Yelp Developers
  2. Sign up or log in with your Yelp account
  3. Create a new app and get your API key
  4. Copy the API key

Step 2: Configure the App

  1. Open app.js in a text editor
  2. Find line 6: YELP_API_KEY: 'YOUR_YELP_API_KEY_HERE'
  3. Replace with your actual API key
  4. Save the file

Step 3: Run the App

Option A: Quick Test (Desktop)

Just open index.html in your browser. Some features may be limited.

Option B: Full HTTPS Server (Recommended)

In your terminal:

npm run dev

Or:

npx live-server --port=3000 --host=0.0.0.0

Option C: Mobile Testing with ngrok

npx serve -s . -l 3000

Then in another terminal:

ngrok http 3000

Use the HTTPS URL on your phone!
